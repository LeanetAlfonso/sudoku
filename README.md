# Sudoku

Sudoku app built in React. Sudoku is logic-based puzzle whose objective is to fill a 9 Ã— 9 grid with digits 1 - 9 so that each column, row, and box contains no repeated values.

Visit [sudokupuzzle.ga](https://www.sudokupuzzle.ga/) to play it now.

<img alt="demo" src="demo.gif" width="400" height="auto"/>

## Running Locally

Clone the repo

```
git clone https://github.com/LeanetAlfonso/sudoku
```

Change directory

```
cd sudoku
```

Install dependencies

```
yarn install
```

Run app

```
yarn start
```
